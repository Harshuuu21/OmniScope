FRONTEND TECHNOLOGY REQUIREMENTS

The generated application must use:

• Next.js 16 App Router
• React 19
• TypeScript
• Tailwind CSS v4
• shadcn/ui
• Radix UI
• Lucide Icons
• Framer Motion
• Motion.dev
• TanStack Query
• Zustand
• React Hook Form
• Zod

Charts

• Recharts for dashboard analytics
• TradingView Lightweight Charts for stock price charts
• Nivo for Treemaps, Sankey, Radar, Heatmaps
• D3.js only for completely custom visualizations
• React Flow for Portfolio X-Ray relationship graphs

Tables

• TanStack Table

Maps

• Mapbox GL

Design Principles

• Use design tokens
• Fully responsive
• WCAG AA accessibility
• Dark Mode first
• Glassmorphism only for overlays
• Never use generic dashboard templates
